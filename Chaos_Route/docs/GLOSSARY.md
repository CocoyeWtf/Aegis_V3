# Glossaire / Glossary

| Terme | EN | Définition |
|-------|-----|------------|
| **PDV** | POS (Point of Sale) | Point de Vente - magasin à livrer |
| **EQP** | Equipment | Équipement/support de livraison (roll, palette, etc.) |
| **RCAM** | Return time | Heure de retour à la base (Return CAM) |
| **SAS** | Airlock | Zone tampon de réception dans un PDV |
| **Base** | Warehouse | Base logistique / entrepôt d'expédition |
| **Tour** | Route/Trip | Tournée de livraison (un véhicule, un trajet) |
| **Tournée** | Route plan | Ensemble des tours d'une journée pour une base |
| **Terme fixe** | Fixed cost | Coût journalier fixe d'un contrat transporteur |
| **Hayon** | Tailgate | Plateforme élévatrice arrière du véhicule |
| **Quai** | Dock | Zone de déchargement du PDV |
| **Mise à quai** | Docking | Temps pour positionner le véhicule au quai |
| **Vitesse commerciale** | Commercial speed | Vitesse moyenne effective (inclut ralentissements) |
| **Distancier** | Distance matrix | Table des distances/durées entre tous les points |
| **SEC** | Dry | Température ambiante (produits secs) |
| **FRAIS** | Fresh | Température réfrigérée (0-4°C) |
| **GEL** | Frozen | Température négative (-18°C) |
| **BI-TEMP** | Bi-temp | Véhicule à deux compartiments température |
| **TRI-TEMP** | Tri-temp | Véhicule à trois compartiments température |
| **RAPIDE** | Fast | Circuit de livraison rapide (J+1) |
| **LENTE** | Slow | Circuit de livraison lente (J+2 ou plus) |
| **Fractionnement** | Split delivery | Division d'une commande PDV sur plusieurs tours |
| **Filiale transport** | Transport subsidiary | Entreprise de transport affiliée au groupe |
